# The Origin of Gurindam

## Overview
The Origin of Gurindam is a 3D action roguelite hack-and-slash game developed in Unity. This project serves as an academic final project focused on integrating Malay cultural preservation with modern game design methodologies. Players assume the role of "The Scholar," navigating a historic fantasy world threatened by a dark fog of forgetfulness. By retrieving shattered fragments of the Gurindam Dua Belas, players fight to preserve the history and culture of the Malay Sultanate.

## Gameplay and Mechanics
The game utilizes a fast-paced combat loop heavily inspired by the action roguelite genre. Death is a core narrative mechanic; upon defeat, the protagonist dissolves into liquid ink and returns to the Sanctuary of the Pen. This hub area allows the player to interact with historical figures like Raja Ali Haji and the Putri Tujuh to exchange Heritage Points for permanent progression upgrades. 

Combat focuses on ink-based mechanics and verbal incantations. 

| Feature | Description |
| :--- | :--- |
| Arbiter's Brush | A heavy staff used for primary melee strikes and ink-based combos. |
| Inscribed Gurindam | Ranged combat system utilizing chanted text as projectiles. |
| Cultural Bosses | Unique encounters featuring mythological entities such as Cindaku and Gajah Mina. |
| Dynamic Environments | Levels include the Overgrown Castles, Burning Bridge, and Flooded Straits. |

## Technical Stack
The project is built on the Unity Engine using C# for all gameplay logic and systems. Visual assets are developed using Blender for 3D modeling, Aseprite for 2D textures, and Adobe Photoshop for post-processing and UI design.
